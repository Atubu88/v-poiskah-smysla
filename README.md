# В поисках смысла

Интерактивный путь о счастье, человеческой нужде, Создателе, Послании, Коране и доверии Богу. Один код работает в обычном браузере и как Telegram Mini App.

## Что входит

- единый маршрут из 8 последовательных этапов;
- 48 интерактивных шагов;
- прямой разговор с пользователем через наблюдения, вопросы, возражения и примеры;
- личная рефлексия без оценки «правильно/неправильно»;
- логические цепочки, один или несколько ответов и раскрывающиеся карточки;
- постепенное открытие этапов и повторное прохождение;
- локальное сохранение прогресса;
- светлая и тёмная темы;
- адаптивный интерфейс для телефона и компьютера;
- Telegram BackButton, системная тема, тактильный отклик и действие «Поделиться»;
- полная работа без сервера, базы данных и Telegram.

## Маршрут

1. Почему нам всегда мало?
2. Ты точно никому не поклоняешься?
3. Когда исчезает контроль.
4. Что говорит этот мир?
5. Кто действительно достоин поклонения?
6. Почему мы должны поверить Посланнику?
7. Вызов, который остаётся.
8. Что значит довериться Богу?

Классической викторины и прежних 10 расследований в этой версии нет.

## Локальный запуск

Нужен Node.js 22.13 или новее.

    npm install
    npm run dev

Vite напечатает локальный адрес. Обычно это http://localhost:3000 или следующий свободный порт.

Проверки:

    npm run lint
    npm run build
    npm run test
    npm run validate:artifact

## Основные файлы

- app/page.tsx — интерфейс, навигация, интерактивы и прогресс;
- app/globals.css — визуальная система и мобильная адаптация;
- lib/journey.ts — восемь этапов и 48 последовательных шагов;
- app/layout.tsx — метаданные и Telegram Web App SDK.

## Хранение данных

Ответы и прогресс сохраняются только на устройстве под ключом meaning-journey-v2. Их можно удалить в разделе «О пути». Серверная база, авторизация и аналитические трекеры не используются.

## Редакторская граница

Новые религиозно содержательные формулировки требуют ручной проверки перед публикацией. Личные ответы пользователя не оцениваются как показатель его веры.
