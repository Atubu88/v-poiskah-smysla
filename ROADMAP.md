# Roadmap

## Перед следующей крупной публикацией

1. Провести квалифицированную богословскую сверку всех формулировок.
2. Пройти весь маршрут и проверить связность переходов.
3. Проверить маршрут на телефоне и компьютере с несколькими пользователями.
4. Утвердить ссылку Telegram-канала, если она понадобится.

## Возможное развитие

1. Добавить короткие аудиофрагменты по этапам.
2. Добавить режим спокойного последовательного прослушивания.
3. После проверки спроса сделать отдельный режим классической викторины.
4. Добавить шведскую и английскую локали.
5. Добавить экспорт личной карты размышлений без передачи ответов на сервер.

## Не входит в текущую версию

Политические ответвления, публичный рейтинг, автоматические богословские ответы, серверный профиль пользователя, реклама и рассылки.
