# Короткая политика конфиденциальности

Ответы, выбранные позиции и прогресс сохраняются только локально в браузере пользователя. Серверная учётная запись, база данных и аналитические трекеры не используются.

При запуске внутри Telegram приложение использует Web App SDK для оформления темы, кнопки возврата, тактильного отклика и системного действия «Поделиться». Приложение не отправляет initData на сервер и не сохраняет Telegram ID.

Пользователь может удалить весь прогресс в разделе «О пути».
