# Инструкция для продолжения разработки

Сначала прочитайте PROJECT_CONTEXT.md, CONTENT_RULES.md, ROADMAP.md и README.md. Не публикуйте сайт и не запускайте Telegram-действия без явного подтверждения владельца.

Критические инварианты:

- содержание развивается только в рамках утверждённого смыслового маршрута;
- не добавлять политические ответвления без отдельного запроса и проверки;
- не придумывать цитаты, аяты, хадисы, номера и внешние источники;
- приложение должно полноценно работать без Telegram и сервера;
- не помещать secrets во frontend, Git или документацию;
- сохранять совместимость Cloudflare Workers;
- любой новый религиозно содержательный текст требует ручной проверки;
- пользовательский интерфейс говорит напрямую и не комментирует внешний источник.

Основные файлы:

- app/page.tsx — приложение, навигация, интерактивы и прогресс;
- app/globals.css — дизайн-система и responsive layout;
- lib/journey.ts — 8 этапов и 48 шагов;
- app/layout.tsx — metadata и Telegram SDK.

Перед передачей выполнить npm run lint, npm run build, npm run test, проверить desktop и mobile, пройти правильный и альтернативный ответ, перезагрузить страницу и убедиться в сохранении прогресса.
