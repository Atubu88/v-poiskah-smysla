# Контекст проекта

## Продукт

«В поисках смысла» — самостоятельный интерактивный путь. Пользователь проходит связанную последовательность: неудовлетворённость земными целями → человеческая нужда → признаки Создателя → критерий истинного Бога → необходимость Посланника → сохраняющийся вызов Корана → доверие Богу.

## Текущие границы

- 8 последовательных этапов;
- 48 коротких шагов;
- прямой разговор без ссылок на внешний текст или рассказчика;
- без классической викторины;
- без прежних 10 расследований;
- без D1, административной панели и серверных данных;
- прогресс хранится локально.

## Аудитория и тон

Люди, размышляющие о смысле жизни и вере. Тон спокойный, уважительный и собеседнический. Каждый этап движется от знакомого наблюдения к вопросу, возможному возражению, примеру и следующему выводу.

## Дизайн

Молочный фон, глубокий зелёный, мягкое золото, крупная книжная типографика, просторные карточки, светлая и тёмная темы. Mobile-first, desktop использует широкую редакционную композицию.

## Архитектура

- Vinext / React / TypeScript → Cloudflare Worker;
- одна клиентская оболочка для браузера и Telegram;
- Telegram включается через feature detection;
- данные этапов находятся в lib/journey.ts;
- прогресс находится в localStorage;
- приложение не требует переменных окружения.
